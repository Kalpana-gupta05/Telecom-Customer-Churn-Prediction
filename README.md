# Telecommunication Customer Churn Prediction

## Project Overview
 Customer churn is a major challenge in the telecommunication industry.
 This project focuses on predicting customer churn in the telecom industry using predictive analytics techniques.

## Objective
 To identify churn-prone customers and support effective customer retention strategies.

## Business Objective 
-Predict customer churn (yes/no)
- Identify key factors influencing churn.
- Support customer retention strategies

## Tools and Technologies used 
- IBM SPSS Modeler
- Predictive Analytics
- Classification Techniques

## Project Contents
 -Telecom churn dataset
 1. telco x modeling data
 2. telco x deployment data
 - SPSS Modeler stream file
 -  Project PDF with screenshots
 - churn Prediction final Report

## Model used
 - Logistic Regression/ Decision Tree (C5.0)
 - Target variable: Customer churn

## Methodology 
1. Data understanding and exploration
2. Data preprocessing and transformation
3. Predictive modeling using classification algorithm
4. Model evaluation and accuracy analysis
5. Business insight generation

## Business Impact
Helps telecom companies identify churn-Prone customer early and take proactive retention action.
